## 平台简介

> 基于ruoyi-plus实现AI聊天和绘画功能-后端

> 本项目完全开源免费！
后台管理界面使用elementUI服务端使用Java17+SpringBoot3.X

实现功能
1. 支持ChatGPT4,Dall-E-3,ChatGPT-4-All,GPTS 模型
2. 支持语音克隆
3. 支持文生图模型
4. 支持微信小程序
5. 支持个人二维码实时到账(易支付)
6. 支持个人微信接入ChatGPT
>
